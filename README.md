# Enterprise Order Orchestrator 🚀

API robusta de orquestación de pedidos construida con **Java 21** y **Spring Boot 3.2.5**. Este proyecto implementa un flujo seguro y resiliente para el procesamiento de transacciones comerciales.

## 🛡️ Características Principales
* **Seguridad:** Implementación de Spring Security con Autenticación Básica.
* **Base de Datos:** Persistencia en PostgreSQL mediante Spring Data JPA.
* **Contenerización:** Infraestructura gestionada con Docker Compose.
* **Documentación:** Swagger UI (OpenAPI 3.0) integrado para pruebas interactivas.
* **Mantenibilidad:** Vigilancia automática de dependencias mediante **GitHub Dependabot**.
* **Observabilidad:** Logging profesional con SLF4J y marcas de estado para trazabilidad.

## 🛠️ Stack Tecnológico
* **Lenguaje:** Java 21 (LTS)
* **Framework:** Spring Boot 3.2.x
* **Base de Datos:** PostgreSQL 15
* **Herramientas:** Docker, Maven, Lombok, Feign Client.
